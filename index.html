About

This page collects 30 of the day's most interesting photos from Flickr and displays them with a parallax effect.

The content changes everyday and I do not decide which photos are used nor which author to show. Each image belongs to its author. 

If you like a particular image and want to see more from the author, you can click on the authors name taking you to their Flickr account.

Technical Information

This project uses Handlebars.js in conjunction with the Flickr API to display the images. To create the scrolling animations and parallax effect, I utilise the powerful combination of the Green Sock Animation Platform and ScrollMagic. This ensures maximal control, browser compatibility and performance for both mobile and desktop devices.
